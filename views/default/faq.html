Frequently Asked Questions





Do I need a Spotify account to use SpotQueue?

Yes you do. In order to use the SpotQueue service, you need to be able to login to a compatible Spotify account.



Is there a mobile version of this website?

There is a mobile version that is accessible from any device.



Are there only specific playlists that I can play?

Once you are logged into your Spotify account, you can choose any songs from any playlist to add to your queue.
